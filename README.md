# הרפתקאות בעברית: משחק תפקידים מבוסס AI

צאו להרפתקה אפית בעולם של פנטזיה, מונחה על ידי בינה מלאכותית. צרו את הדמות שלכם, הרכיבו צוות הרפתקנים, קבלו החלטות גורליות, והשפיעו על הסיפור המתפתח במשחק תפקידים טקסטואלי מלא בעברית, המופעל על ידי מודל Gemini של גוגל.

![תמונה של ממשק המשחק](https://i.imgur.com/u5gCfU7.png)

## ✨ תכונות עיקריות

- **סיפור דינאמי ובלתי צפוי**: כל משחק הוא ייחודי! מנחה המשחק (ה-AI) מגיב לפעולות שלכם ויוצר סיפור מתפתח בזמן אמת.
- **יצירת דמויות עשירה**:
    - בחרו שם, גזע, מקצוע ומין לדמות הראשית שלכם.
    - ה-AI ייצור עבורכם סיפור רקע, נתונים (כוח, זריזות, חוכמה) וציוד התחלתי.
    - **דיוקן דמות ייחודי**: ה-AI יצייר תמונה ייחודית עבור כל דמות שנוצרת!
- **הרכבת קבוצה**: אל תצאו להרפתקה לבד! גייסו עד שלושה בני לוויה למסע, שגם עבורם ה-AI ייצור אישיות ודיוקן.
- **ממשק משתמש נוח**:
    - עקבו אחר הסיפור ביומן משחק אינטראקטיבי.
    - נהלו את הקבוצה שלכם עם דפי דמות מפורטים, הכוללים נקודות חיים, ציוד וסטטיסטיקות.
    - עיצוב רספונסיבי למשחק במחשב או במובייל.
- **שמירה וטעינה**: שמרו את ההתקדמות שלכם וחזרו להרפתקה בכל זמן שתרצו.
- **חוויה מלאה בעברית**: כל ממשק המשתמש, יצירת הדמויות והסיפור מתנהלים לחלוטין בשפה העברית.

## 🚀 איך משחקים?

1.  **יצירת דמות**: במסך הראשי, מלאו את הפרטים עבור הדמות שלכם ולחצו על "צור דמות ראשית". חכו רגע בזמן שה-AI בונה את הגיבור שלכם, כולל סיפור רקע ותמונה.
2.  **הרכבת הקבוצה**: לאחר יצירת הדמות הראשית, תוכלו להוסיף חברים לקבוצה. בחרו את הגזע, המין והמקצוע עבור כל חבר צוות, או החליטו לצאת למסע לבדכם.
3.  **התחלת ההרפתקה**: לחצו על כפתור "התחל!" ומנחה המשחק יציג בפניכם את תמונת הפתיחה של הסיפור.
4.  **ביצוע פעולות**: קראו את תיאורי הסיטואציה. בתיבת הטקסט בתחתית המסך, כתבו בעברית מה הדמות שלכם (או הקבוצה) עושה. למשל:
    - *"אני בודק אם הדלת נעולה"*
    - *"אנחנו תוקפים את הגובלינים!"*
    - *"אני מנסה לשכנע את הסוחר להוריד את המחיר"*
5.  **התקדמות בסיפור**: ה-AI יגיב לפעולה שלכם, יתאר את התוצאות, ויעדכן את מצב הדמויות שלכם (למשל, ירידה בנקודות חיים לאחר קרב).
6.  **שמירה**: בכל שלב, תוכלו ללחוץ על אייקון השמירה כדי לשמור את המשחק. בפעם הבאה שתיכנסו, תוכלו לטעון את ההרפתקה בדיוק מהמקום בו עצרתם.

## 🛠️ טכנולוגיות

- **Frontend**: React, TypeScript
- **AI Model**: Google Gemini API (`@google/genai`)
- **Styling**: Tailwind CSS
- **Storage**: Local Storage
